 Стадия ClassLoader'ы:
Находят класы JvmComperhension, Object, Integer, System и подгружает их. На следующем этапе связывания происходит проверка кода на валидность затем инициализация примитивных статических полей далее подгрузка необходимых классов. Следующий этап инциализация статических блоков кода. В конце данные классов загружаются в метаспэйс.

 Стадия Runtime Data Area и Execution Engine:
0. В стэк добавляется фрейм main() 
1. В фрейм main() добавляется примитивная переменная i со значением 1 
2. В фрейм main() добавляется ссылочная переменная на экземпляр класса Object 
3. В фрейм main() добавляется ссылочная переменная на экземпляр класса Integer со значением 2
4. В стэк добавляется фрейм printAll() с ссылочной переменной o на уже существующий экземпляр класса Object, переменной типа int со значением 1, ссылочной переменной на уже существующий экземпляр класса Integer со значением 2.
